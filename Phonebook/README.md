Запуск

Для запуска приложения выполните следующую команду

bash
python main.py

Использование

После запуска приложения вы увидите меню с доступными действиями:

1. Показать все контакты 
2. Создать контакт 
3. Найти контакт 
4. Изменить контакт 
5. Удалить контакт 
6. Выход

Следуйте инструкциям на экране для выполнения операций.
